로또 번호 생성기 이용약관

시행일: 2025년 1월 1일

제1조 (목적)

본 약관은 로또 번호 생성기(이하 "앱")의 이용 조건 및 절차에 관한 사항을 규정함을 목적으로 합니다.

제2조 (서비스 내용)

앱은 다음과 같은 서비스를 제공합니다:

1. 로또 6/45 번호 랜덤 생성 기능

2. 최근 당첨 통계 기반 번호 추천 기능

3. 번호 제외/포함 필터 기능

4. 최대 5게임 동시 생성 기능

제3조 (면책사항)

1. 앱에서 생성되는 번호는 순수 확률 또는 통계 참고용이며, 당첨을 보장하지 않습니다.

2. 통계 기반 추천 기능은 과거 당첨 데이터를 참고한 것으로, 향후 당첨 확률과 무관합니다.

3. 앱 이용으로 발생하는 금전적 손실에 대해 일체 책임지지 않습니다.

4. 복권 구매는 사용자 본인의 판단과 책임 하에 이루어져야 합니다.

제4조 (당첨 통계 데이터)

1. 통계 데이터는 동행복권 공개 데이터를 기반으로 합니다.

2. 데이터의 정확성을 보장하지 않으며, 지연이나 오류가 발생할 수 있습니다.

제5조 (광고)

앱은 운영을 위해 광고를 포함할 수 있으며, 사용자는 앱 이용 시 노출되는 광고에 동의합니다.

제6조 (약관 변경)

본 약관은 필요 시 변경될 수 있으며, 변경 사항은 앱 내 공지를 통해 안내합니다.
